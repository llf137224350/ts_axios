###ts_axios
```
本项目为使用TypeScript仿写axios，具体实现为：
1、实现了axios({})、axios.get({})、axios.post({})、axios.delete({})、axios.put({})、axios.options({})、
axios.head({})、axios.patch({})以上八中调用方式，使用方式与官方axios一直。
2、支持配置拦截器。
3、支持配置自己的transform
4、支持请求未完成时中断请求
5、支持自定义状态码校验
6、支持url params参数自定义序列化
7、支持withCredentials配置
8、防xsrf攻击
9、支持上传和下载进度监听
```
